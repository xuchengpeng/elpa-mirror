Provides alternatives to `forward-word', `backward-word',
`backward-kill-word', `forward-kill-word' that are generally less
greedy, especially when the backward region contains a newline.

The fundamental purpose of bbww is to overhaul the default Emacs
text editing functionality in a way that comes natural for users
who are used to "normal" modern text editors.

See README.org for information on installation and configuration.
